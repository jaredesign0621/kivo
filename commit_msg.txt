🚀 Release Note: v1.0 - 아이디어 보드 및 리치 텍스트 에디터 업그레이드

✨ 새로운 기능 (New Features)
- 노션(Notion) 스타일 에디터 도입: Tiptap 기반의 강력한 에디터 기능을 추가했습니다. 글쓰기 도중 `/` 키를 눌러 슬래시 명령어 팝업을 호출할 수 있으며, 할 일 목록(체크박스), 인용구 등을 쉽게 삽입할 수 있습니다.
- 이미지 업로드 및 표(Table) 지원: 파이어베이스 스토리지(Firebase Storage)를 연동하여 로컬 이미지를 직접 첨부할 수 있게 되었습니다. 에디터 툴바를 통해 링크, 이미지, 표를 문서 내에 직관적으로 삽입할 수 있습니다.
- 게시물 상세 뷰(Detail View): 2Depth 아이디어 피드에서 특정 게시물을 클릭하면 본문을 쾌적하게 읽을 수 있는 상세 화면으로 진입하는 기능을 구현했습니다.

💡 개선 사항 (Improvements)
- 마우스 오버 액션(Hover Actions): 게시물 목록 피드에서 개별 글에 마우스를 올리면 `[수정]`, `[삭제]` 퀵 버튼이 부드럽게 나타나도록 하여 관리 편의성을 높였습니다.
- 게시물 즉시 수정 기능: 새로운 글 작성 뿐만 아니라, 기존 게시물을 불러와 다시 편집하고 업데이트할 수 있는 폼(Form) 재활용 로직을 구축했습니다.

🎨 UI/UX 업데이트 (Design/UX)
- KIVO 전용 확인 모달(ConfirmModal): 투박한 브라우저 기본 알럿 창(`window.confirm`)을 모두 걷어내고, KIVO 디자인 톤앤매너에 맞춘 세련된 삭제 확인 모달로 100% 교체했습니다.
- 전체 레이아웃 최적화: 모든 LNB 메뉴들의 간격을 통일하고, 메인 뷰어 컨텐츠 영역이 화면 전체 폭(100%)을 넓게 활용하도록 재배치했습니다.
